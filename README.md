## What you’ll find here

- **Fundamentals**: mental models, operating modes (chat/pair/agent/swarms)
- **Prompt & context engineering**: MVC, just-in-time context, compaction, memory
- **Harness engineering**: impact maps → task specs → validation loops
- **Evaluation**: CI gates, regression reduction, benchmarks and pitfalls
- **Cost/performance**: routing, budgets, caching, token/context economy
- **Use cases**: backend, frontend, DevOps/Cloud, automation, embedded AI products

## Scope

This is intentionally **not** a beginner introduction. It focuses on **actionable engineering decisions** and **production-grade workflows**.
